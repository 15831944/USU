Interface Design Theory Lecture 5100.06
  1. Information Organization
    1. Display optimization can improve both efficiency and accuracy. In an experiment, optimization resulted in a decrease of 31% in the time required to complete a task, and 28% in the error rate.
    2. It is a standard principle of information organization to group related information. This can be done by using the principles of proximity, similarity, closure, continuity, and symmetry:
      1. boxing with borders, or even putting information in independent windows, for proximity;
      2. using color to create similarity and contrasting color to create organizational distance;
      3. allowing organizational structure to be completed by the mind with closure; a simple example is creating a solid line with a sequence of dashes -----------------
      4. letting geometric flow give organization, as with a sequence of increasingly smaller rectangles;
      5. placing data in the shape of recognized symmetric figures, typically rectangles, with surrounding blank space.
    3. Where information arranged in columns or sections must be fitted into a scrolling viewport, repeat column headings in each viewport screen. In a similar way, multipage displays should be individually identified.
    4. Narrative information takes 66% more time to understand than structured information.
    5. Crowded information (30% blank surrounding 70% data) takes 45% more time to understand than isolated information (30% data within 70% blank).
    6. Extraneous information slows comprehension.
  2. Presentation
    1. Be sure that presentation is always consistent as to organization, font, color, shape, etc.
    2. Build interfaces which query platforms about aspect ratio, size, colormap limitations, font availablilty, etc., and then react accordingly.
    3. Present numeric data graphically when it is highly voluminous, dynamic, continuous, or used for comparison. Present it textually when it is small, static, discrete, or must be interpreted individually.
    4. Even for expert users, increasing the number of options to select from decreases efficiency. Hick’s Law says selection time is proportional to the log of the number of options.
    5. Encoding options
      1. Size can be used when the larger is more than 50% bigger than the smaller. Typically 3 to 5 sizes can be used effectively.
      2. Up to 20 different shapes (icon profiles) can be used.
      3. Up to 10 different colors, and up to 4 different brightness levels can be used.
      4. Up to 11 different angles can be used.
      5. Line lengths should be limited to 4, widths to 3, and styles (solid, dashed) to 9.
    6. Blink rates should be 2 to 5 hertz, with more than 50% of the cycle displaying the information.
    7. There are a variety of ergonomic standards, for instance, ISO 9241, which give physical measurements and organizations for interaction.
  3. Color
    1. Be conservative, and try to use professionally chosen color schemes.
    2. Always design the interface in monochrome first, so that it will be comprehensible to the 8% of American and West European males who are color blind. This also makes the interface robust on different displays.
    3. Certain combinations - like saturated red and saturated blue, yellow and purple, magenta and green - do not provide comprehensible contrast.
    4. Other combinations - like blue and green, yellow and white, brown and black, green and white - are hard to see.
    5. Some effective contrasts are white and black, blue and yellow, yellow and black.
    6. There may be less color fidelity between displays than you would think.
  4. Text Metrics
    1. Density - Percent of positions filled
    2. Local density - number of positions filled (weighted by distance from the center) in a circle subtending 5 degrees of view
    3. Layout complexity - information theoretic distribution of data by horizontal and vertical distance. This can be minimized by indented rectangular structures.
    4. These metrics can be used to improve both efficiency and user friendliness
    5. The Hawthorne effect must be taken into account. It says that there will be an improvement in subjectively measured satisfaction no matter what you do, because people are pleased to be involved in an experiment, trying something new.
