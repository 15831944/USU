Toolbars and Status Bars Lecture 5100.38

  1. Toolbars - CToolBar
    1. A standard toolbar will be generated by AppWizard if you specify it. It will have new, open, print, etc.
    2. Toolbar buttons typically trigger the same command as a menu item, but custom actions can be defined for actions that don’t respond to menu bar selections.
    3. If you don’t want one of the toolbar buttons generated for you, delete it by dragging it away from the toolbar in the resource editor.
    4. Add a button to your toolbar by:
      1. Drawing its icon in the resource editor. You can trigger the icon editor by clicking on the blank button template.
      2. Match your new button to a command, using the ID found in the property sheet.
      3. Also, add a status message and tool tip in property sheet. The status message will appear on the status bar at the bottom of the application whenever the mouse cursor is over the button, and the tool tip will appear when the cursor lingers for a while. The two are both put into the same text field, separated by a "\n", with the status message first.
  2. StatusBar - CStatusBar
    1. The status bar appears at the bottom of applications. It is made up of panes or indicators, holding text.
    2. SetPaneInfo and GetPaneInfo allow us to determine the id, style, and size of a pane.
    3. Panes also have styles:
      1. SBPS_NOBORDERS   No 3-D border around the pane.
      2. SBPS_POPOUT   Reverse border so that text "pops out."
      3. SBPS_DISABLED   Do not draw text.
      4. SBPS_STRETCH   Stretch pane to fill unused space. Only one pane per status bar can have this style.
      5. SBPS_NORMAL   No stretch, borders, or pop-out.
    4. Message, CAP, NUM, SCRL are some of the standard panes provided by the AppWizard
    5. You can add new panes to your status bar by:
      1. Creating a command ID in the string table resource
      2. Creating a default string or text for the indicator
      3. Adding the ID to the status bar indicators array
      4. Creating a command update handler while will update the text in the pane or indicator when the specified event occurs.
