CS 5100 – Spring, 2002 - Assignment 3

 

Write a Java program which will display a main window containing a menu bar.  The menu bar should provide two drop down menus, File on the left and Help on the right.  The File drop down menu provides two selections: Open (with a Control-O shortcut) and Exit.  The Help drop down menu provides two selections: About.. and Album Files...  Otherwise, the frame encloses a framed blank 500x500 pixel space, with additional space below it to accommodate two buttons and a label.

 

The Open selection presents a file selection dialog which allows the user to open an album file.  Only *.alb files should be displayed in the dialog.  At that point, a label within the frame shows the name of the album file opened, and a button entitled Start appears.  When the user pushes the Start button, its title changes to Next Photo, another button entitled Previous Photo appears, and the first GIF or JPG file listed in the album file appears in the space.  A caption describing the photo also appears below the space.  Photos of different dimensions than 500x500 should be scaled to these dimensions.  From then on, the user can manipulate to Next and Previous buttons to sequence through the GIF or JPG files listed in the album file.  When the last photo is being displayed, the Next Photo button is inactivated.  When the fist photo is being displayed, the Previous Photo button is inactivated.  The buttons should also respond to the cursor control keys left and right arrows.  The user can open another album file and see its contents after having opened a previous album file.

 

The Exit selection causes the program to terminate.

 

The About selection causes a dialog to appear which identifies the author and any other important identifying information about the application.

 

The Album Files selection causes a dialog to appear which identifies the file extension (.alb) of album files and the format of album files read by the application.  That format is:

 

Number of photos listed in album file

First PJG or GIF file containing a photo (absolute path or path relative to the

  directory in which the album file is located)

Caption for first photo.

<Additional pairs of file names and captions>
