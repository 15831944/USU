Suggestions:

 

A conceptual framework might include three object classes: Game, Square, and Side.

 

Side extends button:

    first, second - adjacent Squares

    constructor (x, y, orientation)

    action event handler - changes color, cursor, disables, notifies adjacent

        squares

 

Square extends button:

        top, bottom, left, right - adjacent Sides

        constructor (row, column)

        setSides - constructs/shares sides

        sideChosen - responds to selection of an adjacent side

 

Game:

    area - a JPanel which contains

        scoreArea - a JPanel which contains

            playerLabel

            playerScore

            opponentLabel

            opponentScore

        gameArea - a JPanel which contains

            square - a 3x3 array of Square

        constructor
