CS 5100 - Spring, 2002 - Greg Jones

Assignment 1

 

You will implement a simple "complete the square" game.  It will allow

two players to select sides.  After each selection, if a player completes

a square, then that player can continue to select another side.  If the

player doesn’t complete a square, then the other player will get to select

a side next.

Basic graphical display of the game

    1. The display will show:

        a window completely filled with a scoring area and a game area

        the scoring area will have labels for Player 1 and Player 2,

            and text fields for their scores; (6 points)

        the game area will be a 3x3 matrix of squares. (15 points)

    2. The cursor will be the standard arrow except over the

        sides, where it will be a hand. (9 points)

Basic function of the game

    1. Any unselected side can be selected by a single click:

        it will then change to another color; (6 points)

        it will no longer respond to clicks; (9 points)

        its cursor will become the standard arrow. (6 points)

    2. Selection of a fourth side to a square will result in:

        a change in the color of the square indicating which player made

            the move which completed the square; (12 points)

        an increment in the score of that player.  If a single selection

            completes two squares, two squares will be colored the same and

            two points will be given. (10 points)

Pleasing appearance

    1. Squares, labels, and text fields should be of uniform size. (5 points)

    2. Colors should not be garish. (3 points)

    3. Proportions should not be strange.  In particular, squares should be

        actually square. (9 points)

Ease of use and understanding

    1. The game should not have any additional menus, buttons, shortcuts, or

        other elements that need to be used.  The only user action should be

        the clicking of the mouse over a side. (4 points)

Specified elements

    1. When all the squares have been colored, no further action is possible

        except to click the close button at the upper right of the window

        frame.  At that point the game can be terminated by closing the window,

  with no confirmation dialog. (6 points)
