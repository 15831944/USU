Assignment 2

 

You will extend the "complete the square" game.  It will now permit

one player to play against the program as well as two players against

each other.

Basic graphical display of the game

    1. The display will show:

·         a window completely filled with a scoring area and a game area (15 points)

·         the scoring area will have labels for Player 1 and Player 2 or Opponent, depending on the option chosen; (2 points)

·         and text fields for their scores; (1 point)

·         the game area will be an NxN matrix of squares, where N is chosen by the user and ranges from 3 to 10. (10 points)

Basic function of the game

    1. The user may select the size of the matrix. (5 points)

    2. The user may select the game mode: two-player, simple opponent,

            and challenging opponent. (20 points)

    3. The game will keep track of the best five scores as the percentage

            of squares captured by a player among the total number of

            squares.  This will persist between game executions. (15 points)

Pleasing appearance

    1. Colors can be chosen by the user. (4 points)

    2. Proportions should not be strange.  In particular, squares should be

        actually square. (2 points)

Ease of use and understanding

      1. The game should have file and option menus.  The file menu will have

            selections for New, Exit, and Best Game.  The options menu will

            have selections for Player1 color, Player2 (Opponent) color,

            Side color, and Size of matrix. (10 points)

Specified elements

      1. When the Best Scores selection is made, a dialog will appear showing

            the best five scores.  These will be simple percentages, and no

            player names will be shown. (7 points)

      2. When all the squares have been colored a dialog will appear, indicating

            that the game is over, and stating who won the game.  If the

            winner has a score among the best 5 scores, then the Best Scores

            dialog will also be shown. (9 points)
